# Dioptrika: Identidad De Marca Exacta

## Criterio Principal

Este kit usa como fuente maestra el archivo `assets/master/dioptrika-ai-logo-reference.png`.

No se redibujó, vectorizó ni reinterpretó el logo. Las versiones incluidas son recortes, composiciones o fondos retirados desde la imagen de referencia para conservar el logo tal cual: forma, proporción, espaciado, color, textura y composición.

## Archivos Principales

- `assets/png/logo-principal-original-con-fondo.png`: logo principal con descriptor, recortado desde la referencia.
- `assets/png/logo-principal-original-sin-fondo.png`: mismo logo con fondo retirado.
- `assets/png/logo-horizontal-original-sin-fondo.png`: versión horizontal sin descriptor.
- `assets/png/isologo-original-sin-fondo.png`: símbolo/isologo inferior extraído desde la referencia.
- `assets/png/logo-sin-fondo.png`: alias simple del logo principal sin fondo.
- `assets/png/isologo-sin-fondo.png`: alias simple del isologo sin fondo.
- `assets/png/logo-principal-dark-safe.png`: uso recomendado en modo oscuro sin recolorear el logo.

## Nota Sobre SVG

Los SVG incluidos no son redibujos vectoriales. Son contenedores SVG que incrustan el PNG exacto. Esto se hizo a propósito para no alterar el logo original.

## Modo Claro

Usar el logo sin fondo sobre fondos blancos o muy claros. Para documentos, web y piezas comerciales se recomienda `logo-principal-original-sin-fondo.png`.

## Modo Oscuro

No se recolorea el logo para dark mode porque eso cambiaría el diseño original. La recomendación es usar el logo exacto sobre una superficie blanca o una tarjeta clara dentro del entorno oscuro.

Archivo recomendado: `assets/png/logo-principal-dark-safe.png`.

## Tipografía Recomendada

Para mantener coherencia visual con la referencia:

- Logo: mantener como imagen, no reemplazar por texto.
- Interfaz: Inter.
- Comunicación comercial: Manrope.
- Alternativa del sistema: Segoe UI.

## Usos Correctos

- Usar los PNG extraídos como fuente final del logo.
- Usar el logo con fondo claro en dark mode si se necesita máxima fidelidad.
- Mantener proporciones originales.
- No recrear el wordmark con otra tipografía.
- No cambiar el punto verde de la segunda i.

## Usos Incorrectos

- No vectorizar automáticamente si se busca fidelidad exacta.
- No cambiar colores para dark mode.
- No sustituir el logo por una versión redibujada.
- No deformar, inclinar, aplicar sombras nuevas ni efectos 3D.
- No eliminar el descriptor en aplicaciones donde se haya elegido la versión completa.
