# Dioptrika Brand Kit Exacto

Este kit fue reconstruido usando la imagen de referencia como fuente maestra, sin redibujar ni cambiar el logo.

## Fuente Maestra

`assets/master/dioptrika-ai-logo-reference.png`

## Principio De Uso

Para mantener el logo exactamente como la referencia, usa los PNG como archivos principales. Los SVG incluidos son contenedores con el PNG incrustado, no versiones redibujadas.

## Archivos Recomendados

- `assets/png/logo-principal-original-con-fondo.png`
- `assets/png/logo-principal-original-sin-fondo.png`
- `assets/png/logo-horizontal-original-sin-fondo.png`
- `assets/png/isologo-original-sin-fondo.png`
- `assets/png/logo-sin-fondo.png`
- `assets/png/isologo-sin-fondo.png`
- `assets/png/logo-principal-dark-safe.png`
- `assets/png/avatar-social-light.png`
- `assets/png/avatar-social-dark.png`
- `assets/png/portada-redes-dark.png`
- `assets/png/hero-web-dark.png`

## Dark Mode

No se recolorea el logo. Para dark mode se usa el logo exacto sobre una superficie blanca o clara.
